How to Generate a Favicon for Job Matcher

Option 1: Use an Online Favicon Generator

  1. Go to favicon.io or RealFaviconGenerator
  2. Upload your logo192.png file
  3. Download the generated favicon.ico file
  4. Replace the existing favicon.ico in your public folder with the new one

Option 2: Use Image Editing Software

  1. Open your logo192.png in an image editor like Photoshop, GIMP, or even online tools like Photopea
  2. Resize the image to 32x32 pixels (or 16x16 for smaller favicons)
  3. Save as .ico format
  4. Replace the existing favicon.ico in your public folder

Option 3: Use Command Line Tools

If you have ImageMagick installed:

convert logo192.png -resize 32x32 favicon.ico

After replacing the favicon, restart your development server to see the changes.
